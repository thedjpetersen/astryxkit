# Issue tracker: Local Markdown

Issues and PRDs for this repository live as markdown files in `.scratch/`.

## Conventions

- One feature per directory: `.scratch/<feature-slug>/`
- The PRD is `.scratch/<feature-slug>/PRD.md`
- Implementation issues are `.scratch/<feature-slug>/issues/<NN>-<slug>.md`
- Triage state is a `Status:` line near the top of each issue file.
- Comments append under a `## Comments` heading.

When a skill publishes work to the issue tracker, it creates or updates the corresponding file under `.scratch/`.
